## 验收标准

### 功能验收
- [ ] `go test -count=1 -cover ./...` 整体覆盖率≥50%
- [ ] `golangci-lint run ./...` 无错误
- [ ] `shellcheck scripts/*.sh` 无错误
- [ ] `go test -race ./...` 全部通过
- [ ] `go vet ./...` 无错误
- [ ] `go build ./...` 编译成功
- [ ] `/metrics`和`/health` HTTP端点可用

### 质量验收
- [ ] 无新增竞态条件
- [ ] 无新增内存泄漏
- [ ] 代码复杂度合理（圈复杂度<15）
- [ ] 注释覆盖率>30%
